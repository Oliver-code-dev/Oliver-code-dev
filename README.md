### Sobre mim

Tenho 19 anos, sou do Paraná e divido meu tempo entre a faculdade e o desenvolvimento dos meus próprios projetos de estudo. No momento, meu foco é construir uma base técnica sólida. Em vez de pular direto para frameworks mágicos, estou dedicando tempo para entender como as coisas funcionam por baixo dos panos, desenvolvendo minha lógica e algoritmos principalmente com **C** e **Java**. 

Levo muito a sério a forma como meu código é escrito: procuro manter a organização e criar boas documentações para que qualquer pessoa consiga ler, entender a lógica utilizada e aprender junto comigo. No longo prazo, além de me consolidar no back-end, mantenho vivo o objetivo de aplicar essas habilidades no desenvolvimento de jogos e explorar soluções com inteligência artificial.
